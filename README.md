# Алгоритм кластеризации данных (Он же "k-means")

Это мой проект для лабораторной работы в ВУЗе. 
Он наглядно показывает работу алгоритма.

## Как взаимодействовать с приложением?
После запуска откроется окно.

![image](https://github.com/AlekseiEvRo/k-means/assets/87296876/ad72b8b1-8cf1-4973-bb86-2d9552be489d)

Серая область - зона для визуализации работы алгоритма.
С ней можно взаимодействовать, нажимая ПКМ и ЛКМ в её области, добавляя круги (измерения) и кресты (кластеры).
Также есть кнопки "Выставить точки автоматичеки" и "Выставить кластеры автоматически", 
которые добавляют случайное число кругов и крестов в случайные места в области визуализации.

В общем случае у вас должно получиться что-то вроде этого.

![image](https://github.com/AlekseiEvRo/k-means/assets/87296876/b6f6eb20-07c6-4bc0-8292-aff1c382debc)

Нажимая кнопку "Выполнить шаг алгоритма", можно увидеть, как изменяется цвет кругов и меняется положение крестов.
Если нажать кнопку "Выполнить без остановок", алгоритм отработает целиком и отобразит только конечный результат.
Результат в общем случае будет выглядеть вот так.

![image](https://github.com/AlekseiEvRo/k-means/assets/87296876/d55732c1-312c-41ad-b64f-0d6408fc3d85)
